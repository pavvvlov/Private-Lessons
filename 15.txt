IX,X
1. Изберете правилното наименование за описаната файлова система

[[1] - ext4]

Тази файлова система е скалируемо разширение на друга подобна на нея файлова систена. Често е налична по подразбиране в немалко линукс дистрибуции. Тази файлова система поддържа файл с размери 16 терабайта. Системата поддържа неограничен брой поддиректории.

[[2] - fat32]

Тази файлова система се използва често върху определени устройства. Нейният максимален размер на файл е 4 гигабайта. 

2. Изберете правилното наименование за описаната файлова система

[[1] - ext4]

Тази файлова система е скалируемо разширение на друга подобна на нея файлова систена. Често е налична по подразбиране в немалко линукс дистрибуции. Тази файлова система поддържа файл с размери 16 терабайта. Системата поддържа неограничен брой поддиректории.

[[2] - ntfs]

Тази файлова система се използва по подразбиране в Windows 10. Поддържа файл с размери до 256 TB.

3. Изберете правилното наименование за описаната файлова система

[[1] - ntfs]

Тази файлова система се използва по подразбиране в Windows 10. Поддържа файл с размери до 256 TB.

[[2] - fat32]

Тази файлова система се използва често върху определени устройства. Нейният максимален размер на файл е 4 гигабайта. 

4. За коя файлова система се отнася описанието:

Тя е журнална файлова система. Не се разпознава от Windows 10 по подразбиране, но се използва широко в Линукс.

XI,XII,XIII
1. Демонстрирайте с помощта на команда монтиране на файлова система от устройство /dev/sdb1 на точка /mnt/media
 - sudo mount /dev/sdb1 /mnt/media

2. Демонстрирайте с помощта на команда монтиране на файлова система от устройство /dev/sdc1 на точка /mnt/data
 - sudo mount /dev/sdc1 /mnt/data

3. Демонстрирайте с помощта на команда монтиране на файлова система от устройство /dev/sdb1 на точка /mnt/media, като зададете за тип на файловата система ext4
 - sudo mount -t ext4 /dev/sdb1 /mnt/media

4. Демонстрирайте с помощта на команда монтиране на файлова система от устройство /dev/sdb1 на точка /mnt/media, като зададете за тип на файловата система ntfs
 - sudo mount -t ntfs /dev/sdb1 /mnt/media

5. Демонстрирайте команда, с помощта на която форматирате като ext4 устройството /dev/sdb1
 - sudo mkfs -t ext4 /dev/sdb1

6. Демонстрирайте команда, с помощта на която форматирате като ntfs устройството /dev/sdb1
 - sudo mkfs -t ntfs /dev/sdb1

XVII
1. Пример: По време на теоретичния изпит се предоставя непълен/неработещ/некоректен програмен

фрагмент. Предоставеният фрагмент да се приведе в работещ вид.

Условие:

Създайте чрез команди необходимите директории и файлове от следната  (mkdir, colors, shapes ......)

 - cd ~

mkdir mydir

cd mydir

mkdir colors shapes animals

cd animals

mkdir reptiles mammals

cd ../colors

touch red green blue

echo '#ff0000' > red

echo '#00ff00' > green

echo '#0000ff' > blue

cd ../../

zip -r mydir.zip mydir
